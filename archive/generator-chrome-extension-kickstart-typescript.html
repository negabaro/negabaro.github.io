TypeScript && Chrome Extension 개발환경 간단히 만드는 방법

Posted by on Monday, October 19, 2020 Tags: chrome-extension   1 minute read

yoman,generator-chrome-extension-kickstart-typescript를 이용해 간단히 TypeScript && Chrome Extension 개발환경을 만들어보자.

npm install

 npm install -g yo generator-chrome-extension-kickstart-typescript

커맨드 실행

mkdir project
$ cd project
$ yo chrome-extension-kickstart-typescript

정상적으로 실행되면 아래와 같이 필요에 맞게 항목을 설정해준다.

빌드

정상적으로 생성되었다면 아래 커맨드로 빌드를 실행시켜준다.

npm run dev:chrome

chrome-extenstion에서 로드

chrome://extensions에서 패키지 되지않은 확장파일을 읽어오는 버튼을 클릭해서 project/dist를 로드해준다.
