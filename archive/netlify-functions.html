Netlify Functionsм—җ лҢҖн•ҙ м•Ңм•„ліҙмһҗ

Posted by negabaro kim on Sunday, November 15, 2020 Tags: devops/netlify   5 minute read

Netlify FunctionsмқҙлһҖ?

Netlifyм—җм„ң м ңкіөн•ҳлҠ” м• л“ңмҳЁмӨ‘ н•ҳлӮҳ

FaaSпјҲFunction as a serviceпјүкё°лҠҘмқ„ м ңкіөн•ҙмЈјлҠ” м„ңл№„мҠӨлЎң лӮҙл¶Җм ҒмңјлЎң AWS LambdaлҘј мӢӨн–үн•Ё

Node.js 8.10к°Җ л””нҸҙнҠё мӢӨн–үнҷҳкІҪмқҙлқјкі  н•Ё.

10.xлҘј мӮ¬мҡ©н•ҳкі  мӢ¶мқҖкІҪмҡ°, AWS_LAMBDA_JS_RUNTIMEнҷҳкІҪліҖмҲҳлҘј м§Җм •н•ЁмңјлЎң лІ„м „ліҖкІҪмқҙ к°ҖлҠҘ (н•ҙлӢ№нҷҳкІҪліҖмҲҳлҠ” AWS Lambda runtimeм—җ кё°мһ¬лҗңкІғ)

л№„мҡ©

л¬ҙлЈҢ н”Ңлһңм—җм„ңлҠ” мӣ” 100мӢңк°„ лӮҙм—җ 12л§Ң5000лҰ¬нҖҳмҠӨнҠёк№Ңм§ҖлҠ” л¬ҙлЈҢлқјкі  н•ңлӢӨ.

Lambdaк°Җ мӢӨн–үмӢңк°„мқ„ л§ҘмҠӨмқё 10мҙҲлЎң мһЎмңјл©ҙ 36000лҰ¬нҖҳмҠӨнҠё м»ӨлІ„к°Җ к°ҖлҠҘн•ҳлӢӨкі  ліҙл©ҙлҗңлӢӨ.

к·ё мқҙмғҒмқј кІҪмҡ° мӣ”$25лӢ¬лҹ¬ н”ҢлһңмңјлЎң ліҖкІҪн•ҙмӨҳм•јн•ңлӢӨ.

мҚЁліҙкі  лҠҗлӮҖм җ

AWSм—җ лЎңк·ёмқён•ҳм§Җ м•Ҡкі  lambdaкё°лҠҘмқҙ мӮ¬мҡ©н•ңкұё нҒ° л©”лҰҝмқёкұ° к°ҷлӢӨ.

SPA + FaasнҷҳкІҪкө¬м¶•мӢң Github PagesлҘј мӮ¬мҡ©мӢң lambdaлҘј л”°лЎң л§Ңл“Өм–ҙмЈјлӢӨ ліҙл©ҙ Groupingмқҙ м•Ҳлҗҳм„ң м№ҙмҳӨмҠӨк°Җ лҗҳлҠ” кІҪмҡ°к°Җ л§Һм•ҳлҠ”лҚ° мқҙлҹ° л¶Җл¶„мқҙ к№”лҒ”н•ҙм ём„ң лҢҖл§ҢмЎұ н•ҳкі  мһҲлӢӨ.

SPA мӮ¬мқҙл“ң н”„лЎңм қнҠё м§„н–үмӢң м•һмңјлЎң мһҗмЈј мӮ¬мҡ©н•ҳкІҢ лҗ кұ° к°ҷлӢӨ.

мӮ¬мӣҗмқҙ м•„м§Ғ 97лӘ…мқҙлқјлҠ”лҚ° мқҙм •лҸ„ нҖ„лҰ¬нӢ°к°Җ лӮҳмҳӨлӢӨлӢҲ.. IPOн•ҳл©ҙ м–јлҘё мЈјмӢқмӮ¬кі мӢ¶лӢӨ..

нҷҳкІҪкө¬м¶•

Netlify FunctionsмӢӨн–үмқ„ мң„н•ң нҷҳкІҪкө¬м¶• л°©лІ•м—җ лҢҖн•ҙ м•Ңм•„ліҙмһҗ.

лЁјм Җ NetlifyмҷҖGithubмқ„ м—°кІ°н•ҙмӨҖлӢӨ. мһҗм„ён•ң л°©лІ•мқҖ vue-cliлҘј мӮ¬мҡ©н•ҙ Netlifyм—җ м •м ҒнҺҳмқҙм§ҖлҘј л””н”ҢлЎңмқҙн•ҳлҠ” л°©лІ•мқ„ м°ёкі л°”лһҖлӢӨ.

Netlifyмқҳ м„Өм •нҢҢмқј мһ‘м„ұ

Netlifyмқҳ м„Өм •нҢҢмқјмқёnetlify.tomlнҢҢмқјмқ„ лЈЁнҠёл””л үнҶ лҰ¬м—җ мһ‘м„ұн•ҙмЈјмһҗ.

мқҙ м„Өм •мқҖ NetlifyкҙҖлҰ¬нҺҳмқҙм§Җм—җм„ңлҸ„ м„Өм •мқҙ к°ҖлҠҘн•ҳм§Җл§Ң нӣ„мҲ н•ң лЎңм»¬м—җм„ң н…ҢмҠӨнҠёлҘј мң„н•ҙ мқҙ нҢҢмқјмқҙ н•„мҡ”н•ҳлӢӨ.

#netlify.toml
[build]
  command = "npm run build"
  functions = "dist/functions"

command = Netlifyл””н”ҢлЎңмқҙмӢң мӢӨн–үлҗң м»Өл§Ёл“ң functions = Netlify Functionsмқҳ м—”л“ңнҸ¬мқёнҠё

лЎңм»¬м—җм„ң н…ҢмҠӨнҠё

netlify-lambdaлҘј мӮ¬мҡ©н•ҳл©ҙ лЎңм»¬м—җм„ң Netlify Functionsмқҳ м—җл®¬л ҲмқҙнҠёк°Җ к°ҖлҠҘн•ҳлӢӨ.

мқҙкІҢ м—Ҷмңјл©ҙ ліҖкІҪ -> м»Өл°Ӣ -> л””н”ҢлЎңмқҙ -> лҸҷмһ‘нҷ•мқёмқҙлқјлҠ” н”ҢлЎңмҡ°лҘј л°ҳліөмӢӨн–үн•ҙм•јн•ҳлҜҖлЎң л№„нҡЁмңЁм ҒмқҙлӢӨ.

лЈЁнҠём—җм„ң м•„лһҳ м»Өл§Ёл“ңлЎң netlify-lambdaмқёмҠӨнҶЁ н•ҙмЈјмһҗ.

netlify-lambda install

npm init
npm i netlify-lambda

package.jsonліҖкІҪ

мқёмҠӨнҶЁнӣ„ м•„лһҳмҷҖ к°ҷмқҙ package.jsonмқ„ ліҖкІҪ

мҶҢмҠӨ л””л үнҶ лҰ¬к°Җ src/functions мһҲлӢӨлҠ” м „м ңлӢӨ.

"scripts": {
  "dev": "netlify-lambda serve src/functions",
  "build": "netlify-lambda build src/functions"
}

netlify-lambda serve <folder-name> м»Өл§Ёл“ңлЎң netlify.tomlм—җм„ң м§Җм •н•ң л””л үнҶ лҰ¬лЎң л№Ңл“ң & лЎңм»¬м—җм„ң нҷ•мқёк°ҖлҠҘн•ң м„ңлІ„лҘј мӢӨн–үн•ҙмӨҖлӢӨ.

л””нҸҙнҠё нҸ¬нҠёлҠ” 9000лІҲмқҙкі  вҖ“portлЎң нҸ¬нҠёліҖкІҪмқҙ к°ҖлҠҘ

portм§Җм •мҳҲ

"dev": "netlify-lambda serve resources/api --port 9001"

hello worldмһ‘м„ұ

src/functions/index.jsм—җ м•„лһҳмҷҖ к°ҷмқҖ jsнҢҢмқјмқ„ м¶”к°Җ н•ҙмЈјмһҗ

exports.handler = function(event, context, callback) {
  callback(null, {
    statusCode: 200,
    body: "Hello World"
  });
};

лҸҷмһ‘нҷ•мқё

npm run devлЎң netlify-lambdaлҘј кё°лҸҷн•ҙмЈјкі 

http://localhost:9000/.netlify/functions/index

м—җ м–өмҠӨм„ён•ҳл©ҙ Hello Worldк°Җ н‘ңмӢңлҗҳлҠ”кұё м•Ң мҲҳ мһҲлӢӨ.

лЎңм»¬м—җм„ңмқҳ лҸҷмһ‘нҷ•мқёмқҖ м—¬кё°к№Ңм§ҖлӢӨ.

vue-cliлҘј мӮ¬мҡ©н•ҙ Netlifyм—җ м •м ҒнҺҳмқҙм§ҖлҘј л””н”ҢлЎңмқҙн•ҳлҠ” л°©лІ•к°Җ м„Өм •лҗң м „м ңм—җм„ң мқҙм ң мӢӨм ң netlify functionsм—җ лҸҷмһ‘н•ҳлҠ”м§Җ нҷ•мқён•ҙліҙмһҗ.

м§ҖкёҲк№Ңм§Җмқҳ мҪ”л“ңлҘј githubм—җ commit & push н•ңл’Ө

netlifyмқҳ dashboardк°Җ м•„лһҳмҷҖ к°ҷмқҙ н‘ңмӢңлҗңлӢӨл©ҙ м •мғҒм ҒмңјлЎң л””н”ҢлЎңмқҙк°Җ лҒқлӮңкІғмқҙлӢӨ.

н•ҙлӢ№ functionsмқ„ нҒҙлҰӯн•ҙмЈјкі  endpointм—җ н‘ңмӢңлҗң urlм—җ м–өм„ёмҠӨн•ҙмЈјл©ҙ Function logк°Җ м•„лһҳмІҳлҹј н‘ңмӢңлҗңлӢӨ.(мғҲлЎңкі м№Ё н•„мҡ”)

нҷҳкІҪліҖмҲҳ

API KEYмҷҖ к°ҷмқҙ м„ёнҒҗм–ҙ н•ҳкІҢ кҙҖлҰ¬лҗҳлҠ” м •ліҙлҠ” нҷҳкІҪліҖмҲҳм—җ л„Јм–ҙм„ң кҙҖлҰ¬н•ҙм•јн•  н•„мҡ”к°Җ мһҲлӢӨ.

netlifyм—җм„ң нҷҳкІҪліҖмҲҳ м¶”к°Җ

netlifyлҠ” м•„лһҳ мәЎміҗмҷҖ к°ҷмқҙ dashboardм—җм„ң к°„лӢЁнһҲ нҷҳкІҪліҖмҲҳлҘј м¶”к°Җн•  мҲҳ мһҲлӢӨ.

netlify.tomlм—җ м¶”к°Җн•ҳлҠ”кІғлҸ„ к°ҖлҠҘ

м„ёнҒҗм–ҙ н•ҳм§Җ м•ҠлӢӨл©ҙ netlify.tomlм—җ м§Ғм ‘ л„Јм–ҙмӨҳлҸ„ кҙңм°®лӢӨ.(мҪ”л“ңм—җ нҸ¬н•ЁлҗҳлҜҖлЎң)

netlify.toml

[build.environment]
  NON_SECURE_KEY="test"

м„Өм •н•ң нҷҳкІҪліҖмҲҳлҠ” мҪ”л“ңм—җм„ң process.env.KEYмҷҖ к°ҷмқҖл°©лІ•мңјлЎң к°Җм ёмҳ¬ мҲҳ мһҲлӢӨ.

dotenv-webpack мқёмҠӨнҶЁ

лЎңм»¬м—җм„ң лҸҷмһ‘нҷ•мқёмқ„ мң„н•ҙм„ңлҠ” dotenv-webpackлҘј мқҙмҡ©н•ңлӢӨ.

npm i dotenv-webpack

webpack.functions.jsм¶”к°Җ

const Dotenv = require('dotenv-webpack')

module.exports = {
  plugins: [new Dotenv()],
}

.envм¶”к°Җ

лЈЁнҠём—җ .envнҢҢмқјмқ„ м¶”к°Җн•ҙм„ң н•„мҡ”н•ң нҷҳкІҪліҖмҲҳлҘј м¶”к°Җн•ҙмЈјмһҗ

.env

KEY=value

commandмҲҳм •

м¶”к°Җн•ҙмӨҖ webpackм„Өм •мқ„ м Ғмҡ©н•ҳкё° мң„н•ҙ м•„лһҳмҷҖ к°ҷмқҙ мҲҳм •н•ҙмЈјмһҗ

package.json

"dev": "netlify-lambda serve resources/api --config ./webpack.functions.js",

exports.handler = function(event, context, callback) {

мӢӨн–үнҷҳкІҪмқҙ Nodejs6.10мқё кІҪмҡ°лҠ” мӮ¬мҡ©лӘ»н•ҳлҜҖлЎң мЈјмқҳ
