Rails7 ransack&hotwire을 이용해 검색기능 구현하기

Posted by negabaro kim on Wednesday, May 24, 2023 Tags: rails   2 minute read

ransack

ransack은 검색기능을 간단히 구현가능한 gem이다. hotwire과 같이 사용하면 SPA와 같이 다이나믹하게 구현이 가능

how to install

issue라는 모델에title, content컬럼이 있다는 가정하에 도입순서를 정리해봤다.

Gemfile추가

gem 'ransack'
$ bundle install

controller설정

 def index
    @q = Issue.ransack(params[:q])
    @issues = @q.result.order(id: :desc).page(params[:page]).per(3)
  end

model설정

class Issue < ApplicationRecord
  def self.ransackable_attributes(auth_object = nil)
    ["title", "content"]
  end
end

view설정

<%= search_form_for @q, url: issues_path do |f| %>
    <div class="input-group">
      <%= f.search_field :title_cont, class: "input input-bordered w-full", placeholder: "Search…" %>
      <button class="btn btn-square">
        <svg xmlns="http://www.w3.org/2000/svg" class="h-6 w-6" fill="none" viewBox="0 0 24 24" stroke="currentColor"><path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M21 21l-6-6m2-5a7 7 0 11-14 0 7 7 0 0114 0z" /></svg>
      </button>
    </div>
<% end %>

여기서 포인트는 title_cont부분인데 검색 대상컬럼 뒤에 _cont를 붙이는게 규칙이다.

이 예제에서는 title의 내용을 검색하므로 title_cont로 설정해줬다.

hotwire설정

이 부분은 필수설정은 아니지만 검색한 결과의 내용을 표시하는 부분을 turbo_frame_tag로 묶어주면 SPA처럼 페이지 갱신없이 검색내용을 보여준다.

  <%= turbo_frame_tag 'issues' do %>
    <%= render 'issues' %>
  <% end %>
