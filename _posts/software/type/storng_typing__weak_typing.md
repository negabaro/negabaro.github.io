Python 은 동적타입의 강타입 언어입니다.
그렇다면 강타입이란 무엇일까요?

약타입 언어 : 자료형이 맞지 않을 시에 암묵적으로 타입을 변환하는 언어
강타입 언어 : 자료형이 맞지 않을 시에 에러 발생, 암묵적 변환을 지원하지 않음
예를들어 1 + "1"을 계산하면 약타입 언어인 자바스크립트에서는 "11"로 타입을 변환하여 계산되지만 Python 의 경우에는 TypeError 가 발생하게 됩니다.
