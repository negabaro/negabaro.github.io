

## Firebase apiKey란

Firebase project를 작성하면 `projectId`,`apiKey`,`databaseURL`가 만들어진는데
이것들을 이용해 FireBase의 기능에 접근할 수 있다.

`apiKey`는 `WebApi key`로도 불린다.

api key는  Firebase관리화면 -> 프로젝트 설정 -> WebApi key
