# 인스턴스

인스턴스라는 용어는 객체와 유사하다.
다만 의미상으로 객체는 좀 더 일반적인 의미인 반면 인스턴스는 현재 생성된 바로 그 객체 라는 인스턴트(instant)한 늬앙스가 짙음

# 인스턴스 멤버

객체(인스턴스)마다 가지고 있는 필드(변수)와 메소드를 말함
이들은 각각 인스턴스 필드(변수), 인스턴스 메소드라고 부름

인스턴스 멤버는 객체에 소속된 멤버이기 때문에 객체가 없이는 사용할 수 없음
