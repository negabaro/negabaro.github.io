로미스는 하나의 객체로 미래의 특정 값을 생성하며 3가지의 상태값을 가지고 있습니다.
프로미스의 상태를 이해하고 있어야 사용이 가능합니다.

Pending(대기) : 비동기 처리 로직이 아직 완료되지 않은 상태
Fullfilled(이행) : 비동기 처리가 완료되어 프로미스가 결과값을 반환해준 상태
Rejected(실패) : 비동기 처리가 실패하거나 오류가 발생한 상태
