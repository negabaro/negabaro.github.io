ruby에서 합집합,교집합,여집합을 구현하고 싶을때 각각 `|`,`&`,`-`를 사용해 구현할 수 있다.

## 합집합(|)

```ruby
[1,1,2,3,3] | [3,4,5]
# => [1, 2, 3, 4, 5]
```

양쪽에 중복되는 요소를 제외


## 교집합(&)

```ruby
[1,1,2,3,3] & [3,4,5]
# => [3]
```



양쪽다 존재하는 값만 추출

## 여집합(-)

```ruby
[1,1,2,3,3] - [3,4,5]
=> [1, 1, 2]
```

```ruby





```

왼쪽배열에 존재하지만 오른쪽배열에 존재하면 제외하고
왼쪽배열에 있고 오른쪽배열에 없으면 제외되지 않음.

여집합 = よしゅうごう(余集合) = -

합집합 = わしゅうごう(和集合) = ∪
